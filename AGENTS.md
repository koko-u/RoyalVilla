# Agent Instructions

- Do not modify, create, delete, format, or move any files unless the user explicitly says to do so.
- Investigation, debugging, review, explanation, diagnosis, and "why is this happening?" requests are read-only by default.
- Before making any file change, first present the exact proposed change and ask for confirmation.
- Only edit files after the user explicitly approves with words like "変更して", "修正して", "適用して", "実装して", or "go ahead".
- Running read-only commands is allowed.
- Running commands that generate or modify files requires confirmation first.
